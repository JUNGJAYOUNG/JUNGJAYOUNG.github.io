제어문/배열
==================
제어문
--------
제어문이란, 프로그램의 명령의 흐름을 제어하기 위한 문장을 말함.

1.선택문
  주어진 상황에 따라 실행시켜야 할 명령어를 선택하기 위한 문장을 말함.
  _if문, switch~case문_

2.반복문
  어떠한 조건을 만족할 동안에 어떠한 명령어(들)을 반복하여 실행시키기 위한 문장을 말함. 
  반복문의 종류로는 _for, while, do~while문_이 있다.

3.break와 continue

```
for문의 형식
--------------
a;
b;
for(조건식){
   c;
   d;
}
e;
f;

```
a,b를 수행한 다음에 for문의 조건식을 만족할 동안에 
c,d를 반복하여 수행하고
조건식을 더이상 만족하지 않을때에 for를 탈출하여
e,f를 실행한다.

for문의 조건식은 다음과 같이 표현한다.
```
for(초기값;조건식;증감식){
   명령어(들)
}
```
for() 괄호안에 반드시 세미콜론(;)을 두개를 써야하고
세미콜론 왼쪽에는 반복할 조건식의 초기값을 정해주고
가운데에는 조건식을 써주고
맨 오른쪽에는 증감식을 써줍니다.
그리고 중괄호 안에 그 조건식을 만족할 동안에 반복실행시키고자하는
명령어들을 써 줍니다.

예를 들어 다음은
```
a;
b;
for(int i=1; i<=3; i++){
	c;
	d;
}
e;
f;
```
위의 코드는
a,b를 실행한 다음
i가 1부터 1씩 증가하여 i가 3보다 작거나 같을 동안에
c,와 d를 실행하고
i<=3 라는 조건식을 만족하지 않을때에 반복문을 탈출하여
e,f를 실행한다.
즉 a,b를 실행하고 i가 1일때 c,d를 실행하고
그다음 i가 1증가하여 2일때 c,d를 실행하고
또, i가 1증가하여 3일때 c,d를 실행하고
또, i가 1증가하여 조건식을 판별하니 만족하지 않아 for를 탈출한다.
그리고 e,f를 실행한다.

이때 반복문 안에서 사용되는 변수 i는
반복문 안에서 바로 선언하여 사용할 수도 있고
반복문 바깥에서 미리 선언하여 사용할 수도 있다.

```
int i;
for(int i=1;i<=3;i++){
}
```
이렇게 반복문 바깥에서 미리 변수를 선언하고
반복문 속에서 사용할 수도 있고
```
for(int i=1;i<3;i++){
}
```
이와 같이 반복문 속에서 바로 int i=1과 같이 변수로 선언하여
사용할 수 있다.

변수i가 반복문 속에서만 필요하다면
반복문 속에서 바로 변수를 선언하고
만약, 반복문 바깥에서도 변수i가 필요하다면
반복문이 오기 전에 먼저 변수를 선언해야 한다.

그러나 다음과 같이
반복문이 오기전에 선언한 변수이름과
반복문 속에서 선언한 변수명을 같게하면 오류가 발생합니다.
```
int i;
for(int i=1;i<=3;i++){
}
```
이렇게 하면
반복문이 오기전에 이미 i라는 변수가 있는데
또, 반복문 안에서 변수i를 만들게 되어
오류가 발생한다.

반복문 안에서는
필요하다면 또 다른 반복문이 올 수도 있고
주어진 문제해결을 위하여
선택문 if나 switch문을 표현할 수도 있다.


for문의 형식은 

for(초기값;조건식;증감식){
    명령어
}

와 같이 표현합니다.
초기값이 조건식의 최종값보다 반드시 작을 필요는 없다.
큰값부터 반대로 감소하면서 반복문을 실행시킬 수도 있다.


break
--------
break는 switch문이나 반복문에서 사용되며
어떠한 상황이 되었을때에
switch나 반복문을 탈출하는 용도로 사용합니다.


배열과 반복문
---------------

배열: 자료형이 같은 기억장소의 모임을 배열이라고 한다.
만약에 자료형이 같은 기억장소가 한두개가 아니고 많이 필요하다면
예를 들어 어떤 학교의 전교생이 2000명인데 2000명에 대한 성적처리
하는 프로그램을 작성하려고 합니다.
2000명의 점수를 입력받기 위한 변수를 일일이 2000개를
int student1, student2......student20000
이렇게 변수를 선언하는 것은 불가능한 것은 아니지만 매우 번거로운 일.
변수이름 정하다가 시간 다감.
이와 같이 자료형이 같은 기억장소가 많이 필요할 경우 자료형과 변수를
대표하는 이름과 크기를 정해주어 "배열"을 사용할 수 있다. 
배열은 자료형이 같은 기억장소를 연속하여 확보해준다.
연속하여 확보하기 때문에 차례대로 첫번째 기억장소부터
접근할 수 있는데, 이것을 "인덱스"라고 한다.
예를 들어 정수형변수 5개가 필요하다면 다음과 같이 배열 선언.

int []a = new int[5]
와 같이 배열을 선언
a는 배열이름이고 int[5]는 5개를 만들어라는 이야기다.

이와 같이 배열을 선언하게 되면 5개의 기억장소를 사용할 수 있다.
접근하기 위해서는 배열이름과 인덱스로 접근한다.
예를 들어 맨첫번째 요소에 100 저장
a[0]=100;

a[0]에서 0이 인덱스.
인덱스는 0부터 사용.

만약 배열의 크기가 5이면 사용할수 있는 인덱스는
0,1,2,3,4 까지 5개
따라서
배열의 크기가 n이라면 사용하는 인덱스는
0 ~ n-1

이때 배열의 요소를 모두 출력한다던지
배열의 요소를 모두 더한다던지, 일괄처리하기 위하여
반복문을 사용한다.

또, 배열을 선언함과 동시에
초기화하고자 할때 다음과 같이표현가능.
int []a= {10,20,30};
과 같이 따로 배열의 크기를 정해주지 않고,
오른쪽의 중괄호 {}에 초기값을 컴마(,)로 나열해주면
초기화 한 만큼 배열의 크기가 자동으로 정해진다.
